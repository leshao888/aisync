# Profiles

简体中文 | [English](README.md)

本目录包含 built-in app profiles。

Profiles 必须通过 schema validation，并让安全路径保持默认。

Available profiles：

- `codex.yaml`: stable sync 和 restore profile
- `claude.experimental.yaml`: experimental sync-only profile；restore 已禁用

Planned profiles：

- `cursor.experimental.yaml`: 后续计划支持
